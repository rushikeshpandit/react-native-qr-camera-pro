# react-native-qr-camera-pro

react-native-qr-camera-pro is a lightweight, high-performance React Native QR and barcode scanner built with native Swift and Kotlin using Fabric and TurboModules. It provides fast QR and barcode scanning, torch control, customizable overlay UI, and production-grade camera lifecycle handling with zero unnecessary dependencies. Designed for the New React Native Architecture with strict TypeScript support, clean APIs, and optimized native performance for both iOS and Android

## Installation


```sh
npm install react-native-qr-camera-pro
```


## Usage


```js
import { multiply } from 'react-native-qr-camera-pro';

// ...

const result = multiply(3, 7);
```
